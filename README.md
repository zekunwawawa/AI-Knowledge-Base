# AI 知识大扫盲知识库 🧠
> 个人 AI 学习笔记仓库，基于 Obsidian 管理 + GitHub 同步，专注 AI 名词扫盲与知识沉淀

## 📚 仓库简介
好多名词都不懂，要被名词党逼疯了？这是一个零碎化整理 AI 知识的个人知识库，以「名词扫盲」为核心，覆盖AI，vibe coding,网络基本知识，计算机专业相关如大语言模型、深度学习、Transformer 等核心领域，致力于快速且浅显易懂的扫盲，关注核心，不被名词所困。
- **本地编辑**：Obsidian（支持双向链接、模板快速新建）
- **云端备份**：GitHub（多设备同步 + 版本追溯）
- **内容定位**：从基础概念到进阶原理，逐步构建个人 AI 知识图谱

---

## 🚀 快速使用指南

### 1. 本地浏览（Obsidian）
1. 下载 [Obsidian](https://obsidian.md/)
2. 打开仓库根目录 `my knowledge` 作为 Obsidian Vault
3. 即可浏览、编辑所有笔记，使用模板快速创建新知识点

### 2. 云端同步（GitHub）
#### 新设备拉取仓库
```bash
git clone https://github.com/zekunwawawa/AI-Knowledge-Base.git
```

#### 日常更新流程
1. **添加所有变更**
   ```bash
   git add .
   ```
2. **提交变更（描述本次更新内容）**
   ```bash
   git commit -m "更新：新增【XXX】知识点 / 修正【XXX】笔记"
   ```
3. **推送到 GitHub**
   ```bash
   git push
   ```

#### 拉取最新版本
  ```bash
  git pull
```

---

## 📝 笔记规范
- 每个 AI 名词/知识点对应一篇独立 `.md` 笔记
- 统一使用 `_templates/AI 名词扫盲模板`，包含：
  - 定义
  - 核心原理
  - 应用场景
  - 相关概念（双向链接）
- 支持 Obsidian 标签、搜索、双向链接，便于知识关联检索

## 🛠️ 常用 Git 命令速查
| 操作 | 命令 |
|------|------|
| 查看文件变更状态 | `git status` |
| 添加所有变更到暂存区 | `git add .` |
| 提交变更到本地仓库 | `git commit -m "更新说明"` |
| 推送变更到 GitHub | `git push` |
| 拉取远程最新版本 | `git pull` |
| 撤销暂存区文件 | `git reset HEAD` |
| 放弃本地所有修改 | `git checkout -- .` |
